Разработано с использованием Microsoft Visual Studio 2017.
Версия программы: 1.0 
Системные требования: 
ОС Windows: 7,8,8.1,10. 
Процессор: 1.5GHz 
Оперативная память: 512MB ОЗУ 
Видеокарта: 512MB 
Место на диске: 500МВ 
Необходимые программы и компоненты:
Microsoft Word 8.6.
Microsoft.Office.Interop.Word.dll
.NET Framework 4.6.1
Microsoft SQL Server 2016 LocalDB Версия 13.1.4001.0 и более поздние версии.
Для работы необходимо открыть ServiceCenter\ServiceCenter\bin\Debug\ServiceCenter.exe 
в папке ServiceCenter\ServiceCenter\bin\Debug находятся все необходиме для работы программы данные, в случае перемещения, либо копирования испольняемого файла
необходимо переместить с ним все файлы из папки его расположения, а лучше всего использовать ярлык приложения.
